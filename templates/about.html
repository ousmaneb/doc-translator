About DocTranslator

Professional document translation,
built for accuracy

DocTranslator is an open-source web application that combines DeepL translation, Tesseract OCR, and intelligent format handling into a single seamless workflow.

Our mission

Language should never be a barrier to understanding a document. Whether it's a legal contract, academic paper, medical record, or business report — everyone deserves access to a fast, accurate translation that preserves the document's structure.

DocTranslator combines the best available open-source tools (Tesseract, pdfminer.six, python-docx) with the DeepL translation API — the highest-quality engine for English–French — into a single self-hosted Python application you control completely.
